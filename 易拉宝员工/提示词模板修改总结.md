# 易拉宝提示词模板修改总结

## 修改日期
2026-05-18

## 修改内容

根据您的要求，对易拉宝设计提示词模板进行了以下三项重要修改：

---

## 1. 中部区域：四个重点特点设计

### 修改前
- 每个特点可以有不同的设计形式
- 根据产品类型使用不同形状（水滴形、不规则几何等）
- 可以有旋转、倾斜，增加动感和活力
- 设计较为花哨和多样化

### 修改后
```
- 设计形式: 统一规则的几何形状，简洁专业
  * 推荐形状: 圆角矩形、六边形、圆形、菱形等规则几何图形
  * 4个特点使用相同的形状和设计风格，保持一致性
  * 避免过于花哨或不规则的形状
  * 边框可以有轻微的渐变或光效，但要克制

- 视觉效果:
  * 4个特点框保持完全一致的设计风格
  * 使用柔和的阴影增加层次感（不要过度）
  * 可以有轻微的渐变背景，但要保持简洁
  * 颜色要与背景和主色调协调，确保文字清晰可读
  * 整体风格要专业、规整、不花哨

- 布局:
  * 左侧放置前2个特点，右侧放置后2个特点
  * 4个特点框大小完全一致
  * 位置可以有轻微的高低错落（不超过10%），保持基本对称
  * 间距均匀，留有呼吸感
```

**改进效果：**
- ✅ 四个特点统一规则的几何形状
- ✅ 保持一致性，不花哨
- ✅ 专业、规整的视觉效果

---

## 2. 底部区域：适用场景展示

### 修改前
- 使用插画风格或半写实风格
- 可以使用扁平插画、2.5D插画、等距插画等现代风格
- 色彩与整体风格协调，不要过于写实

### 修改后
```
- 每个场景用写实风格的场景照片+文字标注
- 场景图要求:
  * 使用真实的环境照片风格（写实、清晰、专业）
  * 展示实际的使用场景（如办公室、医院、学校等真实环境）
  * 避免插画、卡通或过度艺术化的风格
  * 照片要有景深效果，突出环境氛围
  * 色彩自然真实，符合实际场景的光线和色调
- 横向排列，均匀分布
- 场景图要简洁明了，一眼能看出使用环境
```

**改进效果：**
- ✅ 使用写实风格的场景照片
- ✅ 移除了插画风格
- ✅ 更加真实、专业

---

## 3. 背景设计：专为净水产品定制

### 修改前
- 根据产品类型（水、科技、健康、商务等）生成不同的背景
- 需要判断产品关键词来决定背景风格
- 多种产品类型的背景模板

### 修改后
```python
def generate_background_description(style_config):
    """
    根据选择的风格生成背景描述（专为净水产品设计）
    
    统一使用水元素背景，融合选择的风格配置
    """
    
    # 提取风格配置
    scene = style_config.get('scene', '现代简约空间')
    lighting = style_config.get('lighting', '柔和自然光')
    colors = style_config.get('colors', '浅蓝色+白色')
    atmosphere = style_config.get('atmosphere', '专业、清爽')

    # 统一的净水产品背景框架，融入选择的风格
    return f"""- 背景主题: 清澈水元素环境（融合{atmosphere}风格）
      * 整体氛围: 流动的水波纹理，清澈透明的水体质感，营造纯净清爽感
      * 场景设定: {scene}
      * 光线效果: {lighting}，光线穿透水体的丁达尔效应
      * 上部区域: 柔和的水面波纹，浅色到白色的自然渐变
      * 中部区域: 水中细腻的气泡上升，水流的动态轨迹，透明感和流动感
      * 下部区域: 水滴飞溅的瞬间，水花的细节，增加动感和活力
      * 色彩方案: {colors}，与水元素自然融合
      * 装饰元素: 可添加模糊的水分子、微小气泡、光斑等细节元素
      * 虚化处理: 背景整体使用高斯模糊和景深效果，明度降低20-30%，确保不抢产品主体"""
```

**改进效果：**
- ✅ 统一使用水元素背景（专为净水公司）
- ✅ 根据选择的风格（科技感/简约商务/自然清新等）融合不同的场景、光线、色彩
- ✅ 不再需要判断产品类型，简化了逻辑

## 优势总结

1. **专业性提升**
   - 四个特点统一规则，更加专业规整
   - 适用场景使用写实照片，更加真实可信

2. **品牌一致性**
   - 所有易拉宝都保留水元素背景，强化净水品牌形象
   - 不同风格通过场景、光线、色彩来区分，而非改变核心元素

3. **代码简化**
   - 移除了复杂的产品类型判断逻辑
   - 统一的背景生成函数，更易维护

4. **灵活性保留**
   - 仍然支持5种不同风格
   - 每种风格都能与水元素背景完美融合

## 后续建议

如果需要进一步调整，可以考虑：

1. **调整四个特点的具体形状**
   - 在提示词中指定具体使用哪种几何形状（如统一使用圆角矩形）

2. **调整场景照片的具体要求**
   - 可以指定照片的角度、景深程度等更细节的要求

3. **调整水元素背景的强度**
   - 可以根据不同风格调整水元素的明显程度

4. **添加更多风格预设**
   - 如需要，可以在 `conversation_state.py` 中添加更多风格配置
