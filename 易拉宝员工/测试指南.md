## ❌ 可能遇到的问题

### 问题1: 找不到图片文件

**错误信息**: `❌ 找不到产品图片: product.jpg`

**解决方法**:
- 确认图片在正确位置
- 检查文件名是否正确(区分大小写)
- 可以修改脚本中的 `PRODUCT_IMAGE_PATH` 为完整路径

### 问题2: API Key错误

**错误信息**: `❌ 请先在脚本中配置 API_KEY`

**解决方法**:
- 打开脚本,确认已修改API Key
- 确认API Key没有多余的空格或引号

### 问题3: 上传失败

**错误信息**: `❌ 上传失败: ...`

**解决方法**:
- 检查网络连接
- 确认API Key有效
- 确认图片格式正确(jpg/png)

### 问题4: 生成失败

**错误信息**: `❌ 生成失败: ...`

**解决方法**:
- 查看错误信息
- 可能是prompt太长或参数不支持
- 可以简化prompt重试

### 问题5: 超时

**错误信息**: `⏰ 超时(超过600秒)`

**解决方法**:
- 可能是服务器繁忙
- 稍后重试
- 或降低分辨率(改为1k)
