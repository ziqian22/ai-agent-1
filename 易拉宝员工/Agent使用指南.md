## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置API密钥

复制 `.env.example` 为 `.env`，并填写你的API密钥：

```env
# Running Hub API配置
RUNNINGHUB_API_KEY=your_api_key_here
RUNNINGHUB_BASE_URL=https://www.runninghub.cn/openapi/v2

# Claude API配置（用于图片分析）
CLAUDE_API_KEY=your_claude_api_key_here
CLAUDE_BASE_URL=https://api.anthropic.com  # 如果使用中转服务，修改此URL

# 结果保存目录
RESULTS_DIR=./results
```

### 3. 启动应用

**方式一：使用启动脚本（推荐）**

双击 `启动Agent.bat` 文件

**方式二：命令行启动**

```bash
streamlit run agent_app.py
```

### 4. 访问界面

浏览器会自动打开 `http://localhost:8501`

## 🎨 功能介绍

### 核心功能

1. **智能图片分析**
   - 使用Claude Opus 4.7 Vision自动识别产品信息
   - 提取产品名称、品牌、特点、适用场景等
   - 支持PNG、JPG格式图片

2. **对话式交互**
   - 自然语言对话，无需复杂操作
   - Agent主动提问，确保信息完整
   - 意图确认，避免误操作

3. **风格选择**
   - 5种预设风格：科技感、简约商务、自然清新、时尚活力、高端奢华
   - 支持自定义风格描述
   - 智能匹配产品特性

4. **实时进度跟踪**
   - 步骤指示器显示当前进度
   - 进度条和百分比
   - 预计剩余时间
   - 详细日志记录

5. **结果展示和下载**
   - 多张设计供选择
   - 支持单张下载
   - 支持批量下载（开发中）

## 📖 使用流程

### 完整流程示意

```
1. 上传产品图片
   ↓
2. AI自动分析产品信息
   ↓
3. 确认产品信息（可修改）
   ↓
4. 选择设计风格
   ↓
5. 确认生成参数
   ↓
6. 开始生成（显示进度）
   ↓
7. 查看结果
   ↓
8. 提供反馈或下载
```

### 详细步骤

#### 步骤1：上传产品图片

1. 点击左侧"上传产品图片"按钮
2. 选择产品宣传图（PNG或JPG格式）
3. 预览图片
4. 点击"🚀 开始分析"

#### 步骤2：确认产品信息

AI会自动分析图片并提取信息：
- 产品名称
- 品牌名称
- 核心卖点
- 产品特点
- 适用场景
- 主要颜色

如果信息有误，可以：
- 点击"✏️ 修改"手动修正
- 点击"🔄 重新分析"重新识别
- 点击"✅ 确认"继续下一步

#### 步骤3：选择设计风格

从5种预设风格中选择：

- **🔷 科技感**：现代、智能、专业
- **💼 简约商务**：高端、克制、精致
- **🌿 自然清新**：健康、环保、舒适
- **⚡ 时尚活力**：活力、年轻、创新
- **👑 高端奢华**：高端、奢华、尊贵

或者输入自定义风格描述。

#### 步骤4：确认生成参数

系统会显示完整的生成参数：
- 产品信息
- 设计风格
- 生成数量
- 主色调
- 尺寸比例

确认无误后点击"✅ 确认生成"。

#### 步骤5：等待生成

右侧会显示实时进度：
- ✅ 分析图片
- ✅ 生成提示词
- ⏳ 调用API
- ⚪ 处理结果

预计需要1-2分钟。

#### 步骤6：查看结果

生成完成后，右侧"生成结果"标签页会显示所有设计。

每张设计有三个操作：
- 👍 喜欢：标记为喜欢
- 💾 下载：下载此设计
- 🔄 重新生成：基于此设计重新生成

#### 步骤7：提供反馈

在对话框中输入反馈：
- "满意"、"很好" → 完成流程
- "重新生成" → 返回风格选择
- 具体建议 → AI根据建议调整

## ❓ 常见问题

### Q1: 如何获取API密钥？

**Running Hub API Key:**
1. 访问 https://www.runninghub.cn
2. 注册并登录
3. 在个人中心获取API Key

**Claude API Key:**
1. 访问 https://console.anthropic.com
2. 注册并登录
3. 创建API Key
4. 如果使用中转服务，需要配置对应的base_url

### Q2: 图片分析不准确怎么办？

1. 确保图片清晰，文字可读
2. 使用高分辨率图片
3. 点击"🔄 重新分析"重试
4. 手动修改识别结果

### Q3: 生成失败怎么办？

1. 检查API密钥是否正确
2. 检查网络连接
3. 查看详细日志了解错误原因
4. 点击"🔄 重试"重新生成

### Q4: 如何修改提示词模板？

编辑 `banner_prompt_template.py` 文件中的 `generate_banner_prompt` 函数。

### Q5: 支持批量生成吗？

当前版本支持单个产品生成多张设计。批量处理多个产品的功能正在开发中（阶段三）。

## 🛠️ 开发说明

### 项目结构

```
易拉宝员工/
├── agent_app.py              # Streamlit界面（新）
├── agent_orchestrator.py     # Agent编排器（新）
├── vision_analyzer.py        # Claude Vision分析器（新）
├── conversation_state.py     # 对话状态定义（新）
├── progress_tracker.py       # 进度跟踪器（新）
├── banner_prompt_template.py # 提示词模板
├── runninghub_client.py      # Running Hub API客户端
├── test_recorder.py          # 测试记录器
├── test_agent.py             # Agent测试脚本（新）
├── requirements.txt          # 依赖列表
├── .env.example              # 环境变量示例
└── 启动Agent.bat             # 启动脚本（新）
```

### 核心模块说明

#### 1. agent_orchestrator.py - Agent编排器

负责：
- 管理对话状态
- 协调各工具调用
- 处理用户输入
- 生成Agent响应

#### 2. vision_analyzer.py - Vision分析器

负责：
- 调用Claude Opus 4.7 Vision API
- 分析产品图片
- 提取结构化信息
- 生成主动提问

#### 3. conversation_state.py - 状态管理

定义：
- 对话状态枚举
- 状态转换规则
- 对话上下文
- 预设风格库

#### 4. progress_tracker.py - 进度跟踪

负责：
- 跟踪任务步骤
- 计算进度百分比
- 估算剩余时间
- 记录详细日志

### 测试
