## 🐛 常见问题排查

### 问题 1：悬浮球不显示

**可能原因：**
- 组件未正确导入
- CSS 样式冲突
- z-index 层级问题

**解决方案：**
1. 检查 App.vue 是否正确导入 HelpAssistant
2. 检查浏览器开发者工具，查看元素是否存在
3. 调整 z-index 值（当前为 9999）

---

### 问题 2：点击悬浮球无反应

**可能原因：**
- JavaScript 错误
- Vue 组件未正确挂载

**解决方案：**
1. 打开浏览器控制台查看错误信息
2. 检查 Vue DevTools 中组件状态
3. 确认 Element Plus 图标库已加载

---

### 问题 3：发送消息后无响应

**可能原因：**
- 后端服务未启动
- API 接口错误
- Claude API Key 无效

**解决方案：**
1. 检查后端是否运行：`http://localhost:8000/`
2. 查看浏览器 Network 标签，检查请求状态
3. 查看后端日志，检查错误信息
4. 验证环境变量配置

---

### 问题 4：助手回答不准确

**可能原因：**
- System Prompt 不够详细
- 模型选择不当

**解决方案：**
1. 检查 main.py 中的 HELP_ASSISTANT_PROMPT
2. 增加更详细的功能说明
3. 考虑使用 Sonnet 模型（更智能，但成本更高）

---

### 问题 5：样式显示异常

**可能原因：**
- CSS 冲突
- 响应式断点问题

**解决方案：**
1. 检查 scoped 样式是否生效
2. 调整面板宽度和高度
3. 测试不同屏幕尺寸
