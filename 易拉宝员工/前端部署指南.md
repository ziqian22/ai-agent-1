# 前端部署指南

## 🚀 方案一：Vercel 部署（推荐，最简单免费）

### 准备工作

首先，你需要知道你的 Railway 后端地址，应该类似：
```
https://your-app-production.up.railway.app
```

### 步骤 1：创建 Vercel 配置文件

在 `易拉宝员工/frontend/` 目录创建 `vercel.json`：

```json
{
  "buildCommand": "npm run build",
  "outputDirectory": "dist",
  "framework": "vite",
  "rewrites": [
    {
      "source": "/(.*)",
      "destination": "/index.html"
    }
  ]
}
```

### 步骤 2：在 GitHub 提交前端代码

确保前端代码已经推送到 GitHub（如果还没推送，需要先解决网络问题）。

### 步骤 3：部署到 Vercel

1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 **"Add New Project"**
4. 选择你的 `ai-agent-1` 仓库
5. 配置项目：
   - **Framework Preset**: Vite
   - **Root Directory**: `易拉宝员工/frontend`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
6. 添加环境变量：
   - Name: `VITE_API_URL`
   - Value: `https://your-railway-backend.railway.app`（你的 Railway 后端地址）
7. 点击 **Deploy**

### 步骤 4：等待部署完成

Vercel 会自动构建并部署，大约 1-2 分钟后完成。

## 🚀 方案二：Railway 部署前端（一起部署）

如果你想前后端都在 Railway：

### 修改 railway.json

将 `易拉宝员工/railway.json` 改为：

```json
{
  "$schema": "https://railway.app/railway.schema.json",
  "build": {
    "builder": "nixpacks"
  },
  "deploy": {
    "startCommand": "cd backend && uvicorn main:app --host 0.0.0.0 --port $PORT",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 10
  }
}
```

### 修改 nixpacks.toml

```toml
[phases.setup]
nixPkgs = ["python311", "nodejs-18_x", "gcc"]

[phases.install]
cmds = [
  "pip install -r requirements.txt",
  "cd frontend && npm install"
]

[phases.build]
cmds = ["cd frontend && npm run build"]

[start]
cmd = "cd backend && uvicorn main:app --host 0.0.0.0 --port $PORT"
```

然后推送到 GitHub，Railway 会自动重新部署。

## 🚀 方案三：本地构建后手动上传

如果网络问题无法推送代码：

### 步骤 1：本地构建

```powershell
cd 易拉宝员工/frontend
npm install
npm run build
```

### 步骤 2：上传到静态托管

构建完成后，`dist` 文件夹包含所有静态文件。你可以：

1. **使用 Netlify Drop**
   - 访问 https://app.netlify.com/drop
   - 直接拖拽 `dist` 文件夹
   - 获得一个临时域名

2. **使用 Cloudflare Pages**
   - 访问 https://pages.cloudflare.com
   - 上传 `dist` 文件夹

### 步骤 3：配置环境变量

在构建前，创建 `易拉宝员工/frontend/.env.production`：

```
VITE_API_URL=https://your-railway-backend.railway.app
```

然后重新构建：

```powershell
npm run build
```

**如果不能推送**：本地构建 + Netlify Drop（最快）
