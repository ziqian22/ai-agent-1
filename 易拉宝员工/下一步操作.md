## 📝 接下来需要做的事（按顺序）

### 1️⃣ 在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 填写仓库信息：
   - **Repository name**: `banner-ai-designer`（或你喜欢的名字）
   - **Description**: `易拉宝AI设计助手 - 基于Claude和RunningHub的智能设计工具`
   - **Visibility**: Private（推荐）或 Public
3. **不要**勾选 "Add a README file"
4. **不要**勾选 "Add .gitignore"
5. **不要**勾选 "Choose a license"
6. 点击 **"Create repository"**

### 2️⃣ 推送代码到 GitHub

GitHub 会显示推送命令，复制并在终端执行：

```bash
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

**或者直接运行：**
```bash
# 替换为你的 GitHub 仓库地址
git remote add origin https://github.com/你的用户名/banner-ai-designer.git
git push -u origin main
```

### 3️⃣ 在 Railway 部署

1. 访问 https://railway.app
2. 用 GitHub 账号登录
3. 点击 **"New Project"**
4. 选择 **"Deploy from GitHub repo"**
5. 选择你刚创建的仓库（`banner-ai-designer`）
6. Railway 会自动开始部署

### 4️⃣ 配置环境变量

在 Railway 项目页面：

1. 点击你的项目
2. 点击 **"Variables"** 标签
3. 点击 **"New Variable"**
4. 添加以下变量：

```
CLAUDE_API_KEY=你的Claude密钥
RUNNINGHUB_API_KEY=你的RunningHub密钥
CLAUDE_BASE_URL=你的Claude地址（可选，如果使用官方API可以不填）
```

5. 点击 **"Add"** 保存
6. Railway 会自动重新部署

### 5️⃣ 获取部署地址

部署完成后（大约 3-5 分钟）：

1. 在 Railway 项目页面，点击 **"Settings"**
2. 找到 **"Domains"** 部分
3. 点击 **"Generate Domain"**
4. 复制生成的域名，例如：`https://banner-ai-designer-production.up.railway.app`

### 6️⃣ 更新前端 API 地址

1. 编辑 `frontend/.env.production` 文件
2. 将 `VITE_API_URL` 改为你的 Railway 域名：
   ```env
   VITE_API_URL=https://banner-ai-designer-production.up.railway.app
   ```
3. 保存并提交：
   ```bash
   git add frontend/.env.production
   git commit -m "更新生产环境 API 地址"
   git push
   ```
4. Railway 会自动重新部署

### 7️⃣ 测试部署

访问你的 Railway 域名，测试功能：
- 上传产品图片
- 生成易拉宝
- 检查所有功能是否正常
