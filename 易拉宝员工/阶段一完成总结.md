## 📝 使用说明

### 如何启动

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **配置API密钥**
复制`.env.example`为`.env`，填写API密钥

3. **启动应用**
双击`启动Agent.bat`或运行：
```bash
streamlit run agent_app.py
```

### 如何测试

```bash
# 测试所有模块
python test_agent.py

# 只测试Vision分析器
python test_agent.py vision

# 只测试Agent流程
python test_agent.py agent
```
