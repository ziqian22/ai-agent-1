# 易拉宝生成系统 - 使用指南

## 📋 快速开始

### 1. 准备文件
- **产品图片**: 放在项目根目录（如 `1_b.png`）
- **品牌Logo**: 放在项目根目录（如 `PUDOW朴道健康水专家-原色.png`）

### 2. 配置产品信息

打开 `test_standard_api.py`，找到 `PRODUCT_INFO` 配置区：

```python
PRODUCT_INFO = {
    "product_name": "你的产品名称",
    "brand": "你的品牌名称",
    "slogan": "核心卖点或口号",
    "features": [
        "特点1",
        "特点2",
        "特点3",
        # ... 添加更多特点
    ],
    "scenes": ["场景1", "场景2", "场景3", "场景4", "场景5"],
    "style": "专业、健康、环保、科技感",  # 或其他风格
    "main_colors": "主色调描述"
}
```

### 3. 配置文件路径

```python
PRODUCT_IMAGE_PATH = "你的产品图片.png"
LOGO_PATH = "你的品牌logo.png"
CUTOUT_PROMPT = "只保留产品主体"  # 根据产品调整
```

### 4. 运行生成

```bash
python test_standard_api.py
```

## 🎨 产品信息配置详解

### product_name (产品名称)
- 完整的产品型号或名称
- 示例: `"勇士K6/K9即开直饮机"`

### brand (品牌名称)
- 品牌全称或定位
- 示例: `"朴道健康水专家"`

### slogan (核心卖点)
- 一句话概括产品核心价值
- 建议: 10-20字，朗朗上口
- 示例: `"澎湃开水持续出 绿色节能新选择"`

### features (产品特点)
- 列表形式，每条一个特点
- 建议: 6-10条
- 突出数字和关键词
- 示例:
  ```python
  "features": [
      "高达125L/H开水量",      # 突出数字
      "100℃开水持续出",        # 突出温度
      "4重深度净化",           # 突出层级
      "LED紫外灯24小时循环杀菌"  # 突出技术
  ]
  ```

### scenes (适用场景)
- 列表形式，3-5个场景
- 会在易拉宝底部生成场景图标
- 示例: `["商务办公", "政府机关", "医院病房", "居委院校", "高铁机场"]`

### style (设计风格)
- 描述整体视觉风格
- 常用选项:
  - `"专业、健康、环保、科技感"` - 适合科技/健康产品
  - `"简约、商务、高端、精致"` - 适合商务产品
  - `"自然、清新、舒适、亲和"` - 适合生活产品
  - `"时尚、活力、年轻、潮流"` - 适合年轻化产品

### main_colors (主色调)
- 描述配色方案
- 示例:
  - `"蓝色+白色+灰色,体现健康水的清洁感"`
  - `"黑色+金色+白色,体现高端商务感"`
  - `"绿色+白色+原木色,体现自然环保"`

## 📂 生成流程

```
1. 上传原始产品图
   ↓
2. 抠图去背景 (可选)
   → 保存到: results/cutout_product.png
   ↓
3. 合成产品图+Logo (可选)
   → 保存到: results/composed_product.png
   → Logo位置: 右上角，避开出血线
   ↓
4. 生成易拉宝
   → 使用自动生成的提示词
   ↓
5. 下载结果
   → 保存到: results/result_0.png
```

## 🎯 提示词模板系统

### 自动生成提示词
系统会根据 `PRODUCT_INFO` 自动生成专业的易拉宝设计提示词，包含：

1. **布局要求**
   - 顶部: 品牌+产品名
   - 中部: 产品图片
   - 底部: 适用场景图标

2. **特点展示**
   - 自动生成2-3列网格布局
   - 提取关键词作为示例
   - 突出数字和核心信息

3. **印刷标准**
   - 3-5cm安全边距
   - 出血位预留
   - 80x200cm标准尺寸

### 手动调整提示词（高级）
如需更精细控制，可以编辑 `banner_prompt_template.py` 中的 `generate_banner_prompt()` 函数。

## 🔧 常见问题

### Q: 生成的易拉宝特点还是纵向排列？
A: 检查 `banner_prompt_template.py` 中的布局示例是否正确生成。可以手动调整提示词强调"横向2-3列"。

### Q: Logo位置不合适？
A: 修改 `compose_product_with_logo()` 函数中的 `safe_margin_x` 和 `safe_margin_y` 参数。

### Q: 适用场景没有生成图标？
A: 这取决于AI模型的理解。可以在提示词中更明确地要求"必须为每个场景生成图标或小图"。

### Q: 如何调整Logo大小？
A: 修改 `compose_product_with_logo()` 中的 `logo_width = int(product_width * 0.15)`，调整 `0.15` 这个比例。

## 💡 最佳实践

1. **产品图片质量**: 使用高清图片（至少2000px宽度）
2. **特点数量**: 6-10条最佳，太多会显得拥挤
3. **场景选择**: 3-5个场景，覆盖主要目标用户群
4. **关键词提取**: 特点中突出数字、技术名词、核心优势
5. **风格一致**: 主色调要与品牌调性匹配

