## 使用方法

### 用户操作流程

1. **上传产品图片**
   - 点击"上传产品图片"
   - 选择产品图（PNG/JPG）
   - 看到产品图预览

2. **上传Logo（可选）**
   - 点击"上传品牌Logo"
   - 选择Logo文件（PNG/JPG）
   - 看到Logo预览（150px）
   - 显示"✅ Logo已上传"

3. **开始分析**
   - 点击"🚀 开始分析"
   - Agent会记录"我上传了产品图片和品牌Logo"

4. **生成易拉宝**
   - 完成对话流程
   - 生成时会自动合成Logo到右上角

5. **移除Logo（可选）**
   - 如果不想要Logo，点击"🗑️ 移除Logo"
   - Logo会被清除

## Logo合成效果

### 位置
- **右上角**
- 距离右边缘：8%（安全边距）
- 距离顶部边缘：6%（安全边距）

### 大小
- Logo宽度：易拉宝宽度的 18%

### 示例
```

Logo宽度：180px (18%)
右边距：80px (8%)
顶部边距：60px (6%)

```

## 技术实现

### 文件保存
```python
# Logo保存路径
temp_uploads/logo_{原文件名}

# 示例
temp_uploads/logo_company_logo.png
```

### 数据流
```
用户上传Logo
    ↓
保存到 temp_uploads/
    ↓
存储路径到 session_state.current_logo
    ↓
传递给 agent.process_message(logo_path=...)
    ↓
保存到 context.logo_path
    ↓
生成时传递给 banner_generator.generate_complete_flow(logo_path=...)
    ↓
调用 compose_with_logo() 合成
    ↓
上传合成后的图片
    ↓
生成易拉宝
```

## 注意事项

### 1. Logo透明度
- 如果Logo有白色背景，建议先处理成透明背景
- 否则会在易拉宝上显示白色方块

### 2. Logo尺寸
- 建议上传高分辨率Logo（至少500px宽）
- 系统会自动缩放到合适大小

### 3. Logo位置
- 固定在右上角
- 自动避开出血线
- 不会遮挡重要内容
