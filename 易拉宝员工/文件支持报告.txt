【支持的文件类型】

1. 图片文件
   - 格式: PNG, JPG, JPEG, GIF, WebP
   - 扩展名: .png, .jpg, .jpeg, .gif, .webp
   - 处理方法: Claude Vision API (analyze_image)
   - Max Tokens: 4096（已优化）
   - 功能:
     * 识别产品图片中的文字（中文+英文）
     * 提取产品名称、品牌、卖点
     * 识别产品特点（可提取10+条）
     * 分析产品类型和适用场景
     * 识别主要颜色
   - 依赖库: Pillow (PIL)
   - 状态: 完全支持

2. PDF 文档
   - 格式: PDF
   - 扩展名: .pdf
   - 处理方法: Claude Document API (analyze_document -> _analyze_pdf)
   - Max Tokens: 3000
   - 功能:
     * 使用 Claude 原生 PDF 支持（type: 'document'）
     * 直接读取 PDF 文档内容（文字+图片）
     * 提取产品信息（同图片）
     * 自动从 PDF 中提取嵌入的图片
     * 支持多页 PDF 文档
   - 依赖库: PyPDF2
   - 状态: 完全支持

3. Word 文档
   - 格式: Word (DOCX/DOC)
   - 扩展名: .docx, .doc
   - 处理方法: python-docx + Claude API (analyze_document -> _analyze_word)
   - Max Tokens: 3000
   - 功能:
     * 提取文档中的所有文本
     * 提取表格中的内容
     * 发送给 Claude 分析产品信息
     * 自动从 Word 中提取嵌入的图片
     * 支持 .docx 和 .doc 格式
   - 依赖库: python-docx
   - 状态: 完全支持

4. PowerPoint 文档
   - 格式: PowerPoint (PPTX/PPT)
   - 扩展名: .pptx, .ppt
   - 处理方法: python-pptx + Claude API (analyze_document -> _analyze_ppt)
   - Max Tokens: 3000
   - 功能:
     * 提取幻灯片中的所有文本
     * 遍历所有幻灯片内容
     * 发送给 Claude 分析产品信息
     * 自动从 PPT 中提取嵌入的图片
     * 支持 .pptx 和 .ppt 格式
   - 依赖库: python-pptx
   - 状态: 完全支持

===============================================================================
【从文件中提取的信息】
===============================================================================

Agent 会从所有文件类型中提取以下信息：

  • product_name: 产品名称（完整型号或名称）
  • brand: 品牌名称
  • slogan: 核心卖点/宣传语
  • features: 产品特点列表（支持10-20条，已优化）
  • product_type: 产品类型（如：饮水机、化妆品等）
  • scenes: 适用场景列表
  • colors: 主要颜色列表

【文件处理流程】
===============================================================================

【图片文件处理流程】
  1. 用户上传图片（PNG/JPG等）
  2. 读取图片并转为 base64
  3. 调用 Claude Vision API 分析
  4. Claude 识别图片中的文字和视觉元素
  5. 返回结构化的产品信息 JSON

【PDF 文件处理流程】
  1. 用户上传 PDF 文档
  2. 读取 PDF 文件并转为 base64
  3. 使用 Claude 原生 PDF 支持（type: 'document'）
  4. Claude 直接读取 PDF 内容（文字+图片）
  5. 返回产品信息 + 使用 PyPDF2 提取嵌入的图片

【Word/PPT 文件处理流程】
  1. 用户上传 Word/PPT 文档
  2. 使用 python-docx/python-pptx 提取文本
  3. 将文本内容（前5000字符）发送给 Claude 分析
  4. Claude 从文本中提取产品信息
  5. 返回产品信息 + 提取嵌入的图片

    - save_to_kb: 是否保存到知识库（可选，默认 false）

【重要说明】
===============================================================================

【Claude 能力】
  - Vision API: 可以识别图片中的文字（中文+英文）
  - Vision API: 可以理解图片的视觉布局和设计
  - Vision API: 可以分析产品的视觉特征和颜色
  - Document API: 原生支持 PDF 文档解析
  - Text Analysis: 可以从提取的文本中分析产品信息

【图片提取】
  - 所有文档类型都支持提取嵌入的图片
  - 提取的图片保存在 temp_uploads/extracted_images/
  - 如果文档中有多张图片，会让用户选择产品图

【注意事项】
  - 图片文件：直接使用 Vision API，效果最好
  - PDF 文档：使用 Claude 原生 PDF 支持，效果好
  - Word/PPT：先提取文本再分析，依赖文本内容质量
  - 文件大小：建议不超过 10MB
  - 文本限制：Word/PPT 提取的文本会被限制在前 5000 字符
