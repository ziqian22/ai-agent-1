## 部署到 Railway

修复完成后，需要部署到 Railway：

```bash
# 1. 提交代码
git add backend/main.py
git commit -m "Fix: 修复知识库导入、图片检查和特点显示问题

- 改进知识库产品图片路径解析，支持多种路径格式
- 添加 product_info 和 product_image 的空值检查
- 移除产品特点数量限制，显示所有特点
- 优化错误提示信息"

# 2. 推送到远程仓库
git push origin main

# 3. Railway 会自动部署
# 等待部署完成后进行测试
```
