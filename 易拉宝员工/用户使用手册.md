## 🎯 系统简介

易拉宝AI设计助手是一款基于人工智能的专业易拉宝设计工具，通过简单的对话交互，即可快速生成高质量的易拉宝设计稿。

### ✨ 核心优势

- **🤖 AI对话交互**：像聊天一样收集需求，无需复杂表单
- **📤 多格式支持**：支持图片、PDF、Word、PPT等多种文件格式
- **🎨 智能设计**：自动提取产品信息，生成专业设计
- **🖼️ 自动抠图**：智能去除背景，突出产品主体
- **💾 知识库管理**：保存常用产品信息，快速复用
- **📊 历史记录**：查看所有生成记录，随时下载

### 🎨 支持的设计风格

1. **科技感** - 现代、智能、清爽的科技风格
2. **简约商务** - 高端、克制、精致的商务风格
3. **自然清新** - 健康、环保、舒适的自然风格
4. **时尚活力** - 活力、年轻、创新的时尚风格
5. **高端奢华** - 高端、奢华、尊贵的奢侈品风格
6. **具体场景** - 真实场景、生活化的场景风格


系统界面分为三个主要区域：

1. **左侧工具栏**
   - 📤 上传文件按钮
   - 📂 知识库按钮
   - 📊 历史记录按钮

2. **中间对话区**
   - 显示与AI助手的对话内容
   - 展示生成的易拉宝图片

3. **底部输入区**
   - 文字输入框
   - 发送按钮

点击左下角的 **📤 上传按钮**，选择以下任一类型的文件：

**支持的文件格式：**
- 图片：PNG、JPG、JPEG
- 文档：PDF、Word (.docx)、PPT (.pptx)

上传后，AI助手会自动分析文件内容并提取产品信息。

## 📱 功能介绍

### 1️⃣ 智能对话生成

通过自然对话的方式收集易拉宝设计所需的信息：

**必需信息：**
- ✅ 产品图片
- ✅ 产品名称
- ✅ 品牌名称

**可选信息（建议提供）：**
- 核心卖点/Slogan
- 产品特点
- 适用场景
- 设计风格
- 主色调

**对话示例：**

```
用户：上传了一张净水器产品图
AI：我分析了您的产品图片，识别到这是一款净水设备...
    请问产品的正式名称是什么？

用户：勇士K9即开直饮机
AI：好的，品牌名称是？

用户：朴道健康水专家
AI：请问产品的核心卖点是什么？比如"即开即饮"、"健康安全"...

用户：澎湃开水持续出，绿色节能新选择
AI：很好！请问有哪些主要特点想要展示？

用户：高达125L/H开水量、100℃开水持续出、4重深度净化
AI：明白了，您想要什么设计风格？
    1. 科技感
    2. 简约商务
    3. 自然清新
    ... (显示全部6种风格)
```

### 2️⃣ 产品知识库

保存常用产品信息，下次生成时可直接调用，无需重复输入。

**如何使用知识库：**

1. **保存产品到知识库**
   - 上传文件时，勾选"保存到知识库"选项
   - 产品信息会永久保存，可随时使用

2. **从知识库选择产品**
   - 点击左侧 📂 知识库按钮
   - 浏览或搜索已保存的产品
   - 点击"使用"按钮，自动加载产品信息

3. **管理知识库**
   - 查看所有已保存的产品
   - 编辑产品信息
   - 删除不需要的产品

**知识库的优势：**
- ⚡ 快速开始：秒级加载产品信息
- 🔄 信息复用：同一产品可生成多种风格
- 📊 统一管理：集中管理所有产品资料

### 3️⃣ 生成历史记录

查看所有历史生成记录，随时下载往期作品。

**功能特点：**
- 📅 按时间倒序显示
- 🔍 快速查找特定产品
- 📥 一键下载所有图片
- 🗑️ 删除不需要的记录

**查看历史记录：**
1. 点击左侧 📊 历史记录按钮
2. 浏览所有生成记录
3. 点击图片可预览大图
4. 点击下载按钮保存到本地

## 📚 使用教程

### 场景一：第一次使用（上传新产品）

**步骤 1：上传产品图片**
```
1. 点击左下角 📤 上传按钮
2. 选择产品图片（推荐使用清晰的产品照片）
3. 等待AI自动分析（约5-10秒）
```

**步骤 2：确认产品信息**
```
AI会展示识别到的信息：
- 产品名称
- 品牌
- 核心卖点
- 产品特点
- 适用场景

您可以：
✓ 确认信息无误
✓ 或告诉AI需要修改的内容
```

**步骤 3：选择设计风格**
```
AI会询问您想要的设计风格，共6种可选：

1. 科技感 - 适合：智能硬件、科技产品、IT设备
2. 简约商务 - 适合：办公用品、企业服务、B2B产品
3. 自然清新 - 适合：健康食品、环保产品、母婴用品
4. 时尚活力 - 适合：运动产品、潮流服饰、年轻化产品
5. 高端奢华 - 适合：奢侈品、高端家电、珠宝首饰
6. 具体场景 - 适合：需要展示实际使用场景的产品

选择"具体场景"时，需要描述场景，例如：
- 商务办公室环境
- 温馨家居客厅
- 北欧风格卧室
```

**步骤 4：补充其他信息**
```
根据对话提示，补充：
- 主色调（如：蓝色+白色）
- 生成数量（默认5张）
- 其他特殊要求
```

**步骤 5：确认并生成**
```
AI会展示完整的信息摘要：
✅ 信息收集完成

让我确认一下收集到的信息：
- 产品名称: 勇士K9即开直饮机
- 品牌: 朴道健康水专家
- 核心卖点: 澎湃开水持续出 绿色节能新选择
- 产品特点:
  1. 高达125L/H开水量
  2. 100℃开水持续出
  3. 4重深度净化
- 设计风格: 科技感
- 生成数量: 5张

请确认以上信息是否正确？
```

回复"确认"或"开始生成"即可开始。

**步骤 6：等待生成**
```
生成过程约需 2-5 分钟，系统会显示进度。

生成包括以下步骤：
1. 智能抠图去背景
2. Logo智能合成
3. AI设计生成
4. 质量优化处理
```

**步骤 7：查看和下载**
```
生成完成后：
- 对话区会显示所有生成的易拉宝图片
- 点击图片可查看大图
- 点击下载按钮保存到本地
- 所有图片会自动保存到历史记录
```

## ❓ 常见问题

### Q1: 上传的图片有什么要求？

**A:** 建议遵循以下标准以获得最佳效果：

- ✅ **格式**：PNG、JPG、JPEG
- ✅ **分辨率**：至少 2000px 宽度
- ✅ **内容**：产品主体清晰，背景简洁
- ✅ **大小**：不超过 10MB

**不建议：**
- ❌ 模糊、低像素的图片
- ❌ 产品被遮挡或不完整
- ❌ 过于复杂的背景

---

### Q2: 生成需要多长时间？

**A:** 时间取决于生成数量和系统负载：

| 生成数量 | 预计时间 |
|---------|---------|
| 1-3张   | 2-3分钟 |
| 4-6张   | 3-5分钟 |
| 7-10张  | 5-8分钟 |

**影响因素：**
- 图片分辨率
- 是否启用抠图
- 系统当前负载

---

### Q3: 可以生成多少张易拉宝？

**A:** 
- 默认每次生成 5 张
- 可自定义生成数量（1-10张）
- 多次生成无限制

**提示：** 一次生成多张可以获得更多设计选择。

### Q4: 生成的易拉宝尺寸是多少？

**A:** 
- **标准尺寸**：80cm × 200cm（易拉宝标准尺寸）
- **分辨率**：2K 高清画质
- **比例**：9:21 竖版

---

### Q5: 如果对生成结果不满意怎么办？

**A:** 您可以：

1. **重新生成**
   - 在对话中说："请重新生成一次"
   - 系统会使用相同参数生成新版本

2. **调整参数**
   - 更换设计风格
   - 调整主色调
   - 修改产品特点描述

3. **提供反馈**
   - 告诉AI具体哪里不满意
   - AI会根据反馈调整设计

**示例对话：**
```
用户：这个版本的配色太暗了
AI：明白了，我会调整为更明亮的配色方案，使用浅蓝色+白色...
```

### Q7: 上传文档后没有识别到产品信息？

**A:** 请检查以下几点：

1. **文档格式是否支持**
   - ✅ PDF、Word (.docx)、PPT (.pptx)
   - ❌ 旧版 Word (.doc) 可能不支持

2. **文档内容是否清晰**
   - 产品信息是否用文字描述
   - 图片是否嵌入文档中
   - 排版是否结构化

3. **尝试其他方式**
   - 单独上传产品图片
   - 通过对话手动输入产品信息

## 📋 附录

### 易拉宝设计规范

**设计要点：**
- ✅ 主要信息放在视线高度（80-150cm）
- ✅ 字体大小：标题 ≥ 60pt，正文 ≥ 30pt
- ✅ 配色不超过 3 种主色
- ✅ 留白充足，避免拥挤

**印刷建议：**
- 分辨率：≥ 300 DPI
- 色彩模式：CMYK
- 文件格式：PNG / PDF

---

### 设计风格详解

#### 1. 科技感
**视觉特点：**
- 配色：蓝色、青色、深灰、白色
- 元素：几何图形、线条、光效、科技纹理
- 字体：无衬线体、现代感强

**适合产品：**
智能硬件、电子产品、软件服务、AI产品、IoT设备

---

#### 2. 简约商务
**视觉特点：**
- 配色：黑、白、灰、深蓝
- 元素：简洁线条、留白充足、对齐工整
- 字体：衬线体或粗体无衬线

**适合产品：**
企业服务、办公用品、金融产品、B2B服务、专业设备

---

#### 3. 自然清新
**视觉特点：**
- 配色：绿色、原木色、浅蓝、米白
- 元素：植物、自然纹理、柔和曲线
- 字体：圆润字体、手写风格

**适合产品：**
健康食品、环保产品、母婴用品、有机产品、养生保健

---

#### 4. 时尚活力
**视觉特点：**
- 配色：明亮对比色、渐变色、荧光色
- 元素：动感线条、年轻化图标、潮流元素
- 字体：粗体、斜体、创意字体

**适合产品：**
运动产品、潮流服饰、年轻化品牌、娱乐产品、创意设计

---

#### 5. 高端奢华
**视觉特点：**
- 配色：黑金、深蓝金、黑白、深色系
- 元素：精致边框、奢华质感、极简主义
- 字体：衬线体、细线体、优雅字体

**适合产品：**
奢侈品、高端家电、珠宝首饰、高端服务、精品酒店

---

#### 6. 具体场景
**视觉特点：**
- 背景：真实场景照片或插画
- 元素：生活化、场景化、沉浸感
- 风格：根据场景调整

**适合产品：**
需要展示使用场景的所有产品

**常用场景：**
- 商务办公室
- 温馨家居
- 咖啡厅
- 健身房
- 户外环境
